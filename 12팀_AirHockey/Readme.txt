12팀 과제 제출물 Readme입니다.

게임 제목: AirHockey

파일 구성은
1. 소스파일
2. 실행파일
3. 리소스파일
4. Project Progress Report
5. 텀프로젝트 진행에 사용한 PM문서

의 5가지로 이루어져 있습니다.

다음은 참고사항입니다.
1. 실행파일은 로컬 환경에서의 실행을 가정하여 ip 설정을 하였습니다. 클라이언트 소스파일의 GameHeader.h 에서 ip를 리모트로 수정하여 빌드하여 리모트 환경에서의 실행이 가능합니다.
2. 리소스는 image 파일들만 사용하며, 클라이언트의 실행 파일에도 존재하므로 별도의 복사 없이 실행 가능합니다.

이상입니다. 읽어주셔서 감사합니다.
